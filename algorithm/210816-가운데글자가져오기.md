## 가운데 글자 가져오기

#### 문제  
단어 s의 가운데 글자를 반환하는 함수, 길이가 짝수라면 가운데 두글자를 반환

#### 제한사항
s는 길이가 1이상, 100이하인 스트링

#### 내 풀이
def solution(s):
    s = list(s)
    
    left, right = 0, len(s) - 1
    #s가 하나일 경우
    if len(s) == 1:
        return s[0]
    #s가 두개일 경우
    if len(s) == 2:
        return s[left] + s[right]
    
    while (left < right):
        left += 1
        right -= 1
        #s가 홀수일 경우
        if left == right:
            return s[left]
        #s가 짝수일 경우
        elif right == (len(s) / 2):
            return s[left] + s[right]

#### 다른 풀이
def solution(s):
    return s[(len(s) - 1) // 2 : len(s) // 2 + 1]
