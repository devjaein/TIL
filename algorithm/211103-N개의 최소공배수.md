## N개의 최소공배수

#### 문제
https://programmers.co.kr/learn/courses/30/lessons/12953

#### 풀이
``` Python3
def solution(arr):
    answer = 1
    def gcd(a, b):
        t = 1
        while t > 0:
            t = a % b
            a, b = b, t
        return a
    
    for i in arr:
        answer *= i // gcd(answer, i)

    return answer
```
