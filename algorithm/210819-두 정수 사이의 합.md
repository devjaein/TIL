## 두 정수 사이의 합

#### 문제 
두 정수 a, b가 주어졌을 때 a와 b 사이에 속한 모든 정수의 합을 리턴하는 함수를 만들어라.

#### 조건
a와 b가 같은 경우 둘 중 아무 수나 리턴
a와 b의 대소관계는 정해져있지 않음

#### 예제
a = 3, b = 5 return 12  
a = 3, b = 3 return 3

#### 내 풀이
``` python
def solution(a, b):
    answer = 0
    # a의 값이 클 경우 
    if (a > b):
        temp = a
        a = b
        b = temp
    
    #a와 b값이 동일할 경우
    if (a == b) :
        return a
    
    for i in range(a, b+1):
        answer += i
    
    return answer
```

#### 다른 풀이

def solution(a, b):
