## 스택 수열

#### 문제
https://www.acmicpc.net/problem/1874

#### 풀이
``` python3
count = int(input())
stack = []
stack_list = []
for i in range(count):
    stack_list.append(int(input()))

for i in range(len(stack_list)-1, -1, -1):
    if stack_list[i-1] > stack_list[i]:
        stack.append(stack_list[i])

print(stack)
```
