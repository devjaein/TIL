## 그룹 단어 체커

#### 문제
https://www.acmicpc.net/problem/1316

#### 풀이
``` python3
n = int(input())
result = n

for i in range(n):
    word = input()
    for j in range(0, len(word) -1):
        if word[j] == word[j+1]:
            pass
        elif word[j] in word[j+1:]:
            result -= 1
            break
print(result)
```
