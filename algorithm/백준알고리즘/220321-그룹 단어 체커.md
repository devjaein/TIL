## 그룹 단어 체커

#### 문제
https://www.acmicpc.net/problem/1316

#### 풀이
``` python3
n = int(input())
group_char = list()
count = 0

for _ in range(n):
    group_char.append(input())

for i in group_char:
    alpha = list()
    for j in range(len(i)):
        if i[j] == i[j+1]:
            pass
        else:
            alpha.append(i[j])
```
